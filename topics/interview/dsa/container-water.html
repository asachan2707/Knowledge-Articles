Container With Most Water

Given n vertical lines represented by heights, find two lines that form a container holding the maximum amount of water.

DifficultyMedium
PatternTwo Pointers
CompaniesGoogle · Amazon · Uber · Meta
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given array height of n non-negative integers, find two indices l and r such that the area min(height[l], height[r]) * (r - l) is maximised.

Example: height = [1,8,6,2,5,4,8,3,7] → 49 (lines at index 1 and 8: min(8,7)×7=49).

Approach

Core Insight & Strategy

Core insight: Start pointers at both ends. Area is bounded by the shorter line. Moving the taller-side pointer inward can only shrink the area (width decreases, height can't increase). Always advance the shorter side to search for a taller partner.

Step-by-step approach

  1. Set left = 0, right = height.length - 1, best = 0.
  2. While left < right: compute area = Math.min(height[left], height[right]) * (right - left).
  3. Update best = Math.max(best, area).
  4. Advance the pointer at the shorter height inward.
  5. Return best.

If we moved L instead: L=2(h=6), R=8: min(6,7)×6 = 36 < 49 ✗ The taller line cannot save us; only the shorter one limits us.
JavaScript Solution

Code

function maxArea(height) {
  let left = 0, right = height.length - 1, best = 0;
  while (left < right) {
    const area = Math.min(height[left], height[right]) * (right - left);
    best = Math.max(best, area);
    if (height[left] <= height[right]) {
      left++;
    } else {
      right--;
    }
  }
  return best;
}

console.log(maxArea([1, 8, 6, 2, 5, 4, 8, 3, 7])); // 49
console.log(maxArea([1, 1]));                         // 1
Time: O(n) Space: O(1)

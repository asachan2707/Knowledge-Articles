Subarray Sum Equals K

Count the number of contiguous subarrays whose elements sum exactly to k using prefix sums.

DifficultyMedium
PatternPrefix Sum + Hash Map
CompaniesGoogle · Meta · Amazon · Microsoft
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given integer array nums and integer k, return the total count of subarrays whose elements sum to k.

Example: nums = [1,2,3], k = 3 → 2 (subarrays [1,2] and [3]).

Approach

Core Insight & Strategy

Core insight: Define prefix sum P[i] = sum of nums[0..i−1]. Subarray nums[j..i−1] sums to P[i] − P[j]. We want P[i] − P[j] = k, so count how many previous prefix sums equal P[i] − k. A Map tracks counts as we go.

Step-by-step approach

  1. Init prefixCount = new Map([[0, 1]]) (base case: empty prefix = 0, seen once).
  2. runningSum = 0, count = 0.
  3. For each num: add to runningSum.
  4. count += prefixCount.get(runningSum - k) ?? 0.
  5. prefixCount.set(runningSum, (prefixCount.get(runningSum) ?? 0) + 1).
Flow Diagram

Trace — nums=[1,2,3], k=3

nums=[1,2,3] k=3 prefixCount = {0:1} runningSum=0 count=0 num=1: runningSum=1 need = 1−3 = −2 not in map → count unchanged prefixCount = {0:1, 1:1} num=2: runningSum=3 need = 3−3 = 0 in map! count += 1 = 1 (subarray [1,2] ✓) prefixCount = {0:1, 1:1, 3:1} num=3: runningSum=6 need = 6−3 = 3 in map! count += 1 = 2 (subarray [3] ✓) prefixCount = {0:1, 1:1, 3:1, 6:1} Answer: 2
JavaScript Solution

Code

function subarraySum(nums, k) {
  const prefixCount = new Map([[0, 1]]); // base case
  let runningSum = 0, count = 0;
  for (const num of nums) {
    runningSum += num;
    count += prefixCount.get(runningSum - k) ?? 0;
    prefixCount.set(runningSum, (prefixCount.get(runningSum) ?? 0) + 1);
  }
  return count;
}

console.log(subarraySum([1, 2, 3], 3));   // 2
console.log(subarraySum([1, 1, 1], 2));   // 2
console.log(subarraySum([-1, -1, 1], 0)); // 1
Time: O(n) Space: O(n)

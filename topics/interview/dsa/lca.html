Lowest Common Ancestor of a BST

Find the lowest node in a BST that is an ancestor of both p and q, exploiting BST ordering for an O(h) iterative solution.

DifficultyMedium
PatternDFS / Recursion (BST ordering)
CompaniesMeta · Google · Microsoft · Amazon
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given a BST and two nodes p and q, find their lowest common ancestor — the deepest node that has both as descendants (a node is a descendant of itself).

Example: root = [6,2,8,0,4,7,9,null,null,3,5], p=2, q=8 → node 6.

Approach

Core Insight & Strategy

Core insight: BST ordering lets us decide direction in O(1). If both p and q are less than the current node, the LCA is in the left subtree. If both are greater, it's in the right subtree. Otherwise, the paths diverge here — this node is the LCA.

Step-by-step approach

  1. Start at node = root.
  2. If both p.val and q.val < node.val: go left.
  3. If both p.val and q.val > node.val: go right.
  4. Otherwise: return node (this is the LCA).
  5. Also provide the general binary tree version using post-order DFS.
Flow Diagram

Trace — BST, p=2, q=8

BST: 6 / \ 2 8 / \ / \ 0 4 7 9 / \ 3 5 At node=6: p.val=2 < 6, q.val=8 > 6 → paths diverge → LCA = 6 ──── If p=2, q=4: ──── At node=6: both 2 and 4 < 6 → go left At node=2: p.val=2 == node.val → LCA = 2 (2 is ancestor of 4) General binary tree (no BST order): Post-order DFS — return node if it equals p or q. If both left and right subtrees return non-null → current = LCA. Otherwise propagate the non-null result up.
JavaScript Solution

Code

class TreeNode {
  constructor(val, left = null, right = null) {
    this.val = val; this.left = left; this.right = right;
  }
}

// BST version — O(h) time, O(1) space (iterative)
function lcaBST(root, p, q) {
  let node = root;
  while (node) {
    if (p.val < node.val && q.val < node.val) {
      node = node.left;
    } else if (p.val > node.val && q.val > node.val) {
      node = node.right;
    } else {
      return node; // paths diverge here → LCA
    }
  }
  return null;
}

// General binary tree version — O(n) DFS
function lcaTree(root, p, q) {
  if (!root || root === p || root === q) return root;
  const left  = lcaTree(root.left, p, q);
  const right = lcaTree(root.right, p, q);
  if (left && right) return root; // p in one subtree, q in the other
  return left || right;
}
Time (BST): O(h) Time (General): O(n) Space: O(h) recursion stack

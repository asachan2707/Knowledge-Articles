Longest Common Subsequence

Find the length of the longest subsequence shared by two strings using a 2-D DP table.

DifficultyMedium
Pattern2-D DP
CompaniesGoogle · Microsoft · Apple
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given strings text1 and text2, return the length of their longest common subsequence. A subsequence does not need to be contiguous.

Example: text1="abcde", text2="ace" → 3 ("ace").

Approach

Core Insight & Strategy

Core insight: dp[i][j] = LCS length of text1[0..i−1] and text2[0..j−1]. If chars match: dp[i][j] = dp[i−1][j−1] + 1. Else: max(dp[i−1][j], dp[i][j−1]). Base: first row and column are 0.

Step-by-step

  1. Create (m+1) × (n+1) dp grid filled with 0.
  2. For i=1..m and j=1..n: if text1[i−1] === text2[j−1]: dp[i][j] = dp[i−1][j−1] + 1. Else dp[i][j] = Math.max(dp[i−1][j], dp[i][j−1]).
  3. Return dp[m][n].

JavaScript Solution

Code

function longestCommonSubsequence(text1, text2) {
  const m = text1.length, n = text2.length;
  const dp = Array.from({ length: m + 1 }, () => new Array(n + 1).fill(0));
  for (let i = 1; i <= m; i++) {
    for (let j = 1; j <= n; j++) {
      if (text1[i - 1] === text2[j - 1]) {
        dp[i][j] = dp[i - 1][j - 1] + 1;
      } else {
        dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
      }
    }
  }
  return dp[m][n];
}

// Space-optimised O(n) — rolling row
function lcsFast(text1, text2) {
  const n = text2.length;
  let prev = new Array(n + 1).fill(0);
  for (const ch1 of text1) {
    const curr = new Array(n + 1).fill(0);
    for (let j = 1; j <= n; j++) {
      curr[j] = ch1 === text2[j - 1]
        ? prev[j - 1] + 1
        : Math.max(prev[j], curr[j - 1]);
    }
    prev = curr;
  }
  return prev[n];
}

console.log(longestCommonSubsequence("abcde", "ace")); // 3
console.log(longestCommonSubsequence("abc", "abc"));   // 3
console.log(longestCommonSubsequence("abc", "def"));   // 0
Time: O(m×n) Space: O(m×n), optimisable to O(n)

Best Time to Buy & Sell Stock

Find the maximum profit achievable from a single buy-then-sell transaction given an array of daily stock prices.

DifficultyEasy
PatternGreedy / Single Pass
CompaniesAmazon · Microsoft · Apple · Google
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given array prices where prices[i] is the stock price on day i, find the maximum profit from buying on one day and selling on a later day. Return 0 if no profit is possible.

Example: prices = [7,1,5,3,6,4] → 5 (buy day 1 at 1, sell day 4 at 6).

Approach

Core Insight & Strategy

Core insight: Scan left to right tracking the minimum price seen so far. At each step, the best profit if selling today is price − minSoFar. Take the running maximum of all these.

Step-by-step approach

  1. Initialize minPrice = Infinity and maxProfit = 0.
  2. For each price, update minPrice = Math.min(minPrice, price).
  3. Update maxProfit = Math.max(maxProfit, price - minPrice).
  4. Return maxProfit.

JavaScript Solution

Code

function maxProfit(prices) {
  let minPrice = Infinity;
  let maxProfit = 0;
  for (const price of prices) {
    minPrice = Math.min(minPrice, price);
    maxProfit = Math.max(maxProfit, price - minPrice);
  }
  return maxProfit;
}

console.log(maxProfit([7, 1, 5, 3, 6, 4])); // 5
console.log(maxProfit([7, 6, 4, 3, 1]));     // 0 (decreasing)
Time: O(n) Space: O(1)

Linked List Cycle

Detect a cycle in a linked list in O(1) space using Floyd's fast-and-slow pointer algorithm, and find the cycle's entry node.

DifficultyEasy
PatternFast & Slow Pointers (Floyd's)
CompaniesAmazon · Microsoft · Apple · Google
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given the head of a linked list, return true if there is a cycle. Bonus: return the node where the cycle begins (or null).

Example: List 3→2→0→-4, with -4 pointing back to 2 → true, cycle starts at node 2.

Approach

Core Insight & Strategy

Core insight: Two pointers on a circular track — a fast runner (moves 2 steps) laps a slow runner (moves 1 step). If they ever meet, there is a cycle. To find the entry: after meeting, reset slow to head and advance both by 1 per step — they meet exactly at the cycle entry.

Step-by-step (detect)

  1. Init slow = fast = head.
  2. While fast and fast.next are non-null: slow = slow.next, fast = fast.next.next. If slow === fast return true.
  3. Return false.

Step-by-step (find entry)

  1. After meeting point found, reset slow = head.
  2. Advance both by 1 until slow === fast. That node is the cycle entry.
Flow Diagram

Trace — 3→2→0→-4→(back to 2)

Nodes: 3(0) → 2(1) → 0(2) → -4(3) → [back to 2(1)] Phase 1 — detect cycle: slow fast Start: 3 3 Step 1: 2 0 (slow+1, fast+2) Step 2: 0 2 (slow+1, fast+2; fast wraps -4→2) Step 3: -4 -4 ← MEET at -4 Phase 2 — find cycle entry: Reset slow=3 (head). fast stays at -4. slow: 3→2 fast: -4→2 Both at 2 ← cycle start Mathematical proof: F = distance head to cycle start C = cycle length, a = distance meeting point to start fast = 2×slow → F+a+nC = 2(F+a) → F = nC-a After reset: slow travels F more steps, fast travels F=nC-a steps (staying in cycle, lands on cycle start). They meet at cycle entry.
JavaScript Solution

Code

class ListNode {
  constructor(val, next = null) { this.val = val; this.next = next; }
}

function hasCycle(head) {
  let slow = head, fast = head;
  while (fast && fast.next) {
    slow = slow.next;
    fast = fast.next.next;
    if (slow === fast) return true;
  }
  return false;
}

function detectCycle(head) {
  let slow = head, fast = head;
  while (fast && fast.next) {
    slow = slow.next;
    fast = fast.next.next;
    if (slow === fast) {
      slow = head;                    // reset to head
      while (slow !== fast) {
        slow = slow.next;
        fast = fast.next;
      }
      return slow;                    // cycle entry node
    }
  }
  return null;
}
Time: O(n) Space: O(1)

Binary Tree Level Order Traversal

Traverse a binary tree level by level (BFS), collecting each level's values into its own array.

DifficultyMedium
PatternBFS / Queue
CompaniesAmazon · Microsoft · Google · Meta
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given the root of a binary tree, return the values of its nodes level by level, left to right, each level as a separate array.

Example: root = [3,9,20,null,null,15,7] → [[3],[9,20],[15,7]].

Approach

Core Insight & Strategy

Core insight: BFS with a queue. At the start of each iteration, snapshot queue.length — that's exactly how many nodes belong to the current level. Process that many nodes, collect their values, enqueue their children, then move to the next level.

Step-by-step approach

  1. If root is null return [].
  2. Init queue = [root], result = [].
  3. While queue is non-empty: snapshot levelSize = queue.length.
  4. Loop levelSize times: dequeue a node, push its value to levelVals, enqueue its non-null children.
  5. Push levelVals to result.
  6. Return result.

JavaScript Solution

Code

class TreeNode {
  constructor(val, left = null, right = null) {
    this.val = val; this.left = left; this.right = right;
  }
}

function levelOrder(root) {
  if (!root) return [];
  const result = [];
  const queue = [root];
  while (queue.length) {
    const levelSize = queue.length; // snapshot — critical
    const levelVals = [];
    for (let i = 0; i < levelSize; i++) {
      const node = queue.shift();
      levelVals.push(node.val);
      if (node.left)  queue.push(node.left);
      if (node.right) queue.push(node.right);
    }
    result.push(levelVals);
  }
  return result;
}
Time: O(n) Space: O(n) — widest level holds up to n/2 nodes

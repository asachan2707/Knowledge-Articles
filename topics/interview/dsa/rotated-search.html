Search in Rotated Sorted Array

Binary search a sorted array that was rotated at an unknown pivot — always one half is sorted, which determines where to search next.

DifficultyMedium
PatternModified Binary Search
CompaniesMeta · Amazon · Microsoft · Google
LanguageJavaScript (ES2020+)
Problem Statement

Description

An integer array was sorted ascending, then rotated at some pivot index. Search for target in O(log n). Return its index or -1 if absent. Values are distinct.

Example: nums = [4,5,6,7,0,1,2], target = 0 → 4.

Approach

Core Insight & Strategy

Core insight: In a rotated sorted array, one half of [left..right] is always fully sorted. Compare nums[mid] with nums[left] to identify which half. Then check if target falls in the sorted half — if yes, binary search there; otherwise search the other half.

Step-by-step

  1. While left <= right: compute mid.
  2. If nums[mid] === target return mid.
  3. If nums[left] <= nums[mid] (left half sorted): if nums[left] <= target < nums[mid] → search left (right = mid-1), else search right (left = mid+1).
  4. Else (right half sorted): if nums[mid] < target <= nums[right] → search right, else search left.
  5. Return -1.

JavaScript Solution

Code

function search(nums, target) {
  let left = 0, right = nums.length - 1;
  while (left <= right) {
    const mid = (left + right) >> 1;
    if (nums[mid] === target) return mid;

    if (nums[left] <= nums[mid]) {        // left half is sorted
      if (nums[left] <= target && target < nums[mid]) {
        right = mid - 1;
      } else {
        left = mid + 1;
      }
    } else {                               // right half is sorted
      if (nums[mid] < target && target <= nums[right]) {
        left = mid + 1;
      } else {
        right = mid - 1;
      }
    }
  }
  return -1;
}

console.log(search([4,5,6,7,0,1,2], 0));  // 4
console.log(search([4,5,6,7,0,1,2], 3));  // -1
console.log(search([1], 0));              // -1
Time: O(log n) Space: O(1)

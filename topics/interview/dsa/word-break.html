Word Break

Determine whether a string can be segmented into valid dictionary words using 1-D DP with early exit.

DifficultyMedium
Pattern1-D DP
CompaniesGoogle · Meta · Amazon · Microsoft
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given a string s and a dictionary wordDict, return true if s can be segmented into a space-separated sequence of one or more dictionary words.

Example: s = "leetcode", wordDict = ["leet","code"] → true ("leet" + "code").

Approach

Core Insight & Strategy

Core insight: dp[i] = true if s[0..i-1] can be segmented. For each end index i, check all start indices j < i: if dp[j] is true and s[j..i-1] is in the dictionary, then dp[i] = true. Base: dp[0] = true (empty string).

Step-by-step

  1. Init wordSet = new Set(wordDict), dp = new Array(n+1).fill(false), dp[0] = true.
  2. For i from 1 to n: for j from 0 to i-1: if dp[j] && wordSet.has(s.slice(j, i)) → set dp[i] = true and break.
  3. Return dp[n].

JavaScript Solution

Code

function wordBreak(s, wordDict) {
  const wordSet = new Set(wordDict);
  const n = s.length;
  const dp = new Array(n + 1).fill(false);
  dp[0] = true; // empty string base case

  for (let i = 1; i <= n; i++) {
    for (let j = 0; j < i; j++) {
      if (dp[j] && wordSet.has(s.slice(j, i))) {
        dp[i] = true;
        break; // early exit once found
      }
    }
  }
  return dp[n];
}

console.log(wordBreak("leetcode", ["leet", "code"]));       // true
console.log(wordBreak("applepenapple", ["apple", "pen"]));  // true
console.log(wordBreak("catsandog", ["cats", "dog", "sand", "and", "cat"])); // false
Time: O(n² × L) where L = avg word length for slice Space: O(n + dict size)

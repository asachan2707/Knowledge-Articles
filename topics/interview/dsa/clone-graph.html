Clone Graph

Deep-copy a connected undirected graph by DFS, using a Map to track cloned nodes and handle cycles.

DifficultyMedium
PatternDFS + Hash Map
CompaniesMeta · Google · Amazon
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given a reference to a node in a connected undirected graph, return a deep copy. Each node has val and neighbors (array of Node references).

Example: Input node 1 in graph 1–2–3–4–1 → returns a completely new graph with the same structure.

Approach

Core Insight & Strategy

Core insight: Use a Map from originalNode → cloneNode. When DFS visits a node already in the map, return the existing clone (handles cycles). Create the clone before recursing on neighbours to prevent infinite loops.

Step-by-step

  1. If node is null return null.
  2. If visited.has(node) return visited.get(node).
  3. Create clone = new Node(node.val).
  4. Store visited.set(node, clone) before recursing.
  5. For each neighbour, push dfs(neighbour) to clone.neighbors.
  6. Return clone.
Flow Diagram

(creates clone3, recurses to clone2 and clone4) c2.neighbors = [clone1, clone3] ✓ neighbour 4: similar c1.neighbors = [clone2, clone4] ✓ Key: store clone BEFORE recursing to break cycles.
JavaScript Solution

Code

class Node {
  constructor(val, neighbors = []) {
    this.val = val;
    this.neighbors = neighbors;
  }
}

function cloneGraph(node) {
  if (!node) return null;
  const visited = new Map(); // original → clone

  function dfs(n) {
    if (visited.has(n)) return visited.get(n);
    const clone = new Node(n.val);
    visited.set(n, clone); // store BEFORE recursing (handles cycles)
    for (const neighbour of n.neighbors) {
      clone.neighbors.push(dfs(neighbour));
    }
    return clone;
  }

  return dfs(node);
}
Time: O(V+E) Space: O(V) — visited map + recursion stack

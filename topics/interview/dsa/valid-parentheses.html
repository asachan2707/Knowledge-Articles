Valid Parentheses

Validate bracket sequences with a stack — push opens, pop and match on every close.

DifficultyEasy
PatternStack
CompaniesAmazon · Google · Microsoft · Meta
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given string s containing only (, ), {, }, [, ], determine if it is valid. Valid means every open bracket is closed by the same type in the correct order.

Example: s = "({[]})" → true. s = "([)]" → false.

Approach

Core Insight & Strategy

Core insight: Use a stack. For every opening bracket push it. For every closing bracket, check that the top of the stack is the matching opener — if not (or stack is empty), invalid. At the end the stack must be empty.

Step-by-step

  1. Create matching = { ')':'(', '}':'{', ']':'[' } and empty stack = [].
  2. For each char ch: if it's a closing bracket, check stack.at(-1) === matching[ch]. If yes pop; else return false. Otherwise push.
  3. Return stack.length === 0.

JavaScript Solution

Code

function isValid(s) {
  const matching = { ')': '(', '}': '{', ']': '[' };
  const stack = [];
  for (const ch of s) {
    if (ch in matching) {
      if (stack.at(-1) !== matching[ch]) return false;
      stack.pop();
    } else {
      stack.push(ch);
    }
  }
  return stack.length === 0;
}

console.log(isValid("()[]{}"));  // true
console.log(isValid("({[]})"));  // true
console.log(isValid("([)]"));    // false
console.log(isValid("{[]"));     // false
Time: O(n) Space: O(n)

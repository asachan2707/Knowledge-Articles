Coin Change

Find the minimum number of coins needed to make a target amount — the classic unbounded knapsack DP.

DifficultyMedium
Pattern1-D DP (Unbounded Knapsack)
CompaniesAmazon · Google · Microsoft
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given an integer array coins (coin denominations) and integer amount, return the fewest coins needed to make up amount. Return -1 if it cannot be made.

Example: coins = [1,5,6,9], amount = 11 → 2 (5+6).

Approach

Core Insight & Strategy

Core insight: dp[a] = minimum coins to make amount a. For each amount, try every coin: dp[a] = min(dp[a], dp[a−coin] + 1). Base: dp[0] = 0. Fill from 1 to amount.

Step-by-step

  1. Create dp = new Array(amount + 1).fill(Infinity).
  2. Set dp[0] = 0.
  3. For each amount a from 1 to amount: for each coin: if coin <= a update dp[a] = Math.min(dp[a], dp[a - coin] + 1).
  4. Return dp[amount] === Infinity ? -1 : dp[amount].

JavaScript Solution

Code

function coinChange(coins, amount) {
  const dp = new Array(amount + 1).fill(Infinity);
  dp[0] = 0;
  for (let a = 1; a <= amount; a++) {
    for (const coin of coins) {
      if (coin <= a) {
        dp[a] = Math.min(dp[a], dp[a - coin] + 1);
      }
    }
  }
  return dp[amount] === Infinity ? -1 : dp[amount];
}

console.log(coinChange([1, 5, 6, 9], 11)); // 2
console.log(coinChange([1, 2, 5], 11));     // 3
console.log(coinChange([2], 3));             // -1
Time: O(amount × coins.length) Space: O(amount)

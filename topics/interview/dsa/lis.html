Longest Increasing Subsequence

Find the length of the longest strictly increasing subsequence — solved with O(n²) DP or O(n log n) patience sorting.

DifficultyMedium
Pattern1-D DP / Binary Search
CompaniesGoogle · Amazon · Microsoft
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given an integer array nums, return the length of the longest strictly increasing subsequence.

Example: nums = [10,9,2,5,3,7,101,18] → 4 (e.g. [2,3,7,101]).

Approach

Core Insight & Strategy

Core insight: O(n²) DP: dp[i] = LIS length ending at index i. For each j < i where nums[j] < nums[i], dp[i] = max(dp[i], dp[j]+1). O(n log n): maintain a tails array where tails[k] = smallest tail of all increasing subsequences of length k+1. Binary search to place each number.

Step-by-step

  1. O(n²): init dp = Array(n).fill(1). For i from 1..n-1: for j from 0..i-1: if nums[j] < nums[i]: dp[i] = Math.max(dp[i], dp[j]+1). Return Math.max(...dp).
  2. O(n log n): init tails = []. For each num: binary search for leftmost index where tails[pos] >= num. If not found, push. Else replace tails[pos] = num. Return tails.length.

JavaScript Solution

Code

// O(n^2) — clear and usually sufficient
function lisDP(nums) {
  const dp = new Array(nums.length).fill(1);
  for (let i = 1; i < nums.length; i++) {
    for (let j = 0; j < i; j++) {
      if (nums[j] < nums[i]) dp[i] = Math.max(dp[i], dp[j] + 1);
    }
  }
  return Math.max(...dp);
}

// O(n log n) — patience sorting with binary search
function lisFast(nums) {
  const tails = [];
  for (const num of nums) {
    let lo = 0, hi = tails.length;
    while (lo < hi) {          // find leftmost tails[pos] >= num
      const mid = (lo + hi) >> 1;
      if (tails[mid] < num) lo = mid + 1;
      else hi = mid;
    }
    tails[lo] = num;           // extend or replace
  }
  return tails.length;
}

console.log(lisDP([10,9,2,5,3,7,101,18]));   // 4
console.log(lisFast([10,9,2,5,3,7,101,18])); // 4
Time: O(n²) or O(n log n) Space: O(n)

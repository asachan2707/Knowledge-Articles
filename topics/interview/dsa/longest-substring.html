Longest Substring Without Repeating Characters

Find the length of the longest substring that contains no duplicate characters using a sliding window.

DifficultyMedium
PatternSliding Window
CompaniesGoogle · Meta · Uber · Amazon
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given string s, return the length of the longest substring with all unique characters.

Example: s = "abcabcbb" → 3 ("abc").

Approach

Core Insight & Strategy

Core insight: Maintain a window [left, right]. Expand right each step. If the new character already appears inside the window, jump left past its previous occurrence. A Map stores the most recent index of each character so the jump is O(1).

Step-by-step approach

  1. Create charIndex = new Map() (char → last seen index), left = 0, maxLen = 0.
  2. For each right from 0 to s.length−1: let ch = s[right].
  3. If charIndex has ch and its stored index >= left, move left = charIndex.get(ch) + 1.
  4. Update charIndex.set(ch, right).
  5. Update maxLen = Math.max(maxLen, right - left + 1).

JavaScript Solution

Code

function lengthOfLongestSubstring(s) {
  const charIndex = new Map(); // char → most recent index
  let left = 0, maxLen = 0;
  for (let right = 0; right < s.length; right++) {
    const ch = s[right];
    if (charIndex.has(ch) && charIndex.get(ch) >= left) {
      left = charIndex.get(ch) + 1; // jump past duplicate
    }
    charIndex.set(ch, right);
    maxLen = Math.max(maxLen, right - left + 1);
  }
  return maxLen;
}

console.log(lengthOfLongestSubstring("abcabcbb")); // 3
console.log(lengthOfLongestSubstring("bbbbb"));    // 1
console.log(lengthOfLongestSubstring("pwwkew"));   // 3
Time: O(n) Space: O(min(n, charset))

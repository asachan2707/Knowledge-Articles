Course Schedule

Determine if all courses can be finished given prerequisite pairs — equivalent to detecting a cycle in a directed graph.

DifficultyMedium
PatternTopological Sort / DFS Cycle Detection
CompaniesGoogle · Meta · Uber · Amazon
LanguageJavaScript (ES2020+)
Problem Statement

Description

There are numCourses labeled 0 to n−1. Given prerequisites as pairs [a, b] meaning "must take b before a", return true if it is possible to finish all courses.

Example: numCourses=2, prerequisites=[[1,0]] → true. prerequisites=[[1,0],[0,1]] → false (cycle).

Approach

Core Insight & Strategy

Core insight: Model as a directed graph. The question becomes: does the graph contain a cycle? Use DFS with three states: 0 = unvisited, 1 = in current DFS path (visiting), 2 = fully processed (done). A back-edge (visiting a node already in the current path) signals a cycle.

Step-by-step

  1. Build adjacency list from prerequisites.
  2. Create state = new Array(numCourses).fill(0).
  3. For each unvisited node call hasCycle(node).
  4. hasCycle: if state[node] === 1 return true (back edge). If state[node] === 2 return false (already done). Set state[node] = 1, recurse on neighbours. Set state[node] = 2, return false.
  5. If any call returns true, return false (cannot finish).
Flow Diagram

Trace — numCourses=4, prerequisites=[[1,0],[2,0],[3,1],[3,2]]

Graph (directed edges b→a meaning "b is prereq for a"): 0 → 1 0 → 2 1 → 3 2 → 3 state[i]: 0=unvisited, 1=visiting, 2=done hasCycle(0): state[0]=1 (visiting) hasCycle(1): state[1]=1 hasCycle(3): state[3]=1 no unvisited neighbours → state[3]=2 ✓ state[1]=2 ✓ hasCycle(2): state[2]=1 neighbour 3: state[3]=2 (done) → skip, no cycle state[2]=2 ✓ state[0]=2 ✓ No cycle detected → return true (can finish) Cycle example — add edge [0,3] (0 depends on 3): hasCycle(0): state[0]=1 hasCycle(3): state[3]=1 hasCycle(0): state[0] === 1 → CYCLE! return false
JavaScript Solution

Code

function canFinish(numCourses, prerequisites) {
  const graph = Array.from({ length: numCourses }, () => []);
  for (const [a, b] of prerequisites) graph[b].push(a);

  // 0=unvisited, 1=visiting, 2=done
  const state = new Array(numCourses).fill(0);

  function hasCycle(node) {
    if (state[node] === 1) return true;  // back edge
    if (state[node] === 2) return false; // already processed
    state[node] = 1;
    for (const neighbour of graph[node]) {
      if (hasCycle(neighbour)) return true;
    }
    state[node] = 2;
    return false;
  }

  for (let i = 0; i < numCourses; i++) {
    if (state[i] === 0 && hasCycle(i)) return false;
  }
  return true;
}

console.log(canFinish(2, [[1,0]]));        // true
console.log(canFinish(2, [[1,0],[0,1]]));  // false

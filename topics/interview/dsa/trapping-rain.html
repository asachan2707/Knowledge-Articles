Trapping Rain Water

Given an elevation map, compute the total units of rainwater that can be trapped between the bars.

DifficultyHard
PatternTwo Pointers
CompaniesGoogle · Amazon · Microsoft · Meta
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given array height of n non-negative integers representing an elevation map, compute how much water can be trapped after rain.

Example: height = [0,1,0,2,1,0,1,3,2,1,2,1] → 6.

Approach

Core Insight & Strategy

Core insight: Water at bar i = min(maxLeft, maxRight) − height[i]. Use two pointers to avoid precomputing max arrays. The side with the smaller running max determines the water level at its pointer position — we don't need the other side's exact max.

Step-by-step approach

  1. Set left=0, right=height.length-1, leftMax=0, rightMax=0, water=0.
  2. While left < right: if leftMax <= rightMax, process left pointer — update leftMax, add leftMax - height[left] to water, advance left.
  3. Otherwise process right pointer — update rightMax, add rightMax - height[right] to water, advance right.
  4. Return water.

JavaScript Solution

Code

function trap(height) {
  let left = 0, right = height.length - 1;
  let leftMax = 0, rightMax = 0, water = 0;
  while (left < right) {
    if (leftMax <= rightMax) {
      leftMax = Math.max(leftMax, height[left]);
      water += leftMax - height[left];
      left++;
    } else {
      rightMax = Math.max(rightMax, height[right]);
      water += rightMax - height[right];
      right--;
    }
  }
  return water;
}

console.log(trap([0,1,0,2,1,0,1,3,2,1,2,1])); // 6
console.log(trap([4,2,0,3,2,5]));               // 9
Time: O(n) Space: O(1)

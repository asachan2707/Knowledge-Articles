Reverse a Linked List

Reverse a singly linked list iteratively in O(1) space and recursively, by re-wiring three pointers per step.

DifficultyEasy
PatternPointer Manipulation
CompaniesAmazon · Meta · Microsoft · Apple
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given the head of a singly linked list, reverse the list and return the new head.

Example: 1 → 2 → 3 → 4 → 5 → null → 5 → 4 → 3 → 2 → 1 → null

Approach

Core Insight & Strategy

Core insight: Keep three pointers: prev (already reversed), curr (current node), next (saved before overwrite). Each step: save curr.next, flip curr.next = prev, advance both prev and curr.

Step-by-step

  1. Init prev = null, curr = head.
  2. While curr !== null: save next = curr.next. Set curr.next = prev. Advance prev = curr, curr = next.
  3. Return prev (new head).

JavaScript Solution

Code

class ListNode {
  constructor(val, next = null) { this.val = val; this.next = next; }
}

// Iterative — O(1) space
function reverseList(head) {
  let prev = null, curr = head;
  while (curr !== null) {
    const next = curr.next;
    curr.next = prev;
    prev = curr;
    curr = next;
  }
  return prev;
}

// Recursive — O(n) call stack
function reverseListRec(head) {
  if (!head || !head.next) return head;
  const newHead = reverseListRec(head.next);
  head.next.next = head;
  head.next = null;
  return newHead;
}
Time: O(n) Space: O(1) iterative, O(n) recursive

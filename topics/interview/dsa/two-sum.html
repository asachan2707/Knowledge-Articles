Two Sum

Given an integer array nums and a target, return the indices of the two numbers that add up to the target. Exactly one solution is guaranteed.

DifficultyEasy
PatternHash Map (complement lookup)
CompaniesAmazon · Google · Meta · Microsoft
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given an array of integers nums and an integer target, return the indices [i, j] such that nums[i] + nums[j] === target. You may not use the same element twice. Exactly one valid answer exists.

Example: nums = [2, 7, 11, 15], target = 9 → [0, 1] because nums[0] + nums[1] = 2 + 7 = 9.

Approach

Core Insight & Strategy

Core insight: Instead of scanning every pair (O(n²)), store each number's index in a Map as you scan. For each new number, check if its complement (target − num) is already in the map. One pass, O(n).

Step-by-step approach

  1. Create an empty Map seen (value → index).
  2. Iterate nums with index i.
  3. Compute complement = target - nums[i].
  4. If seen has complement, return [seen.get(complement), i].
  5. Otherwise store seen.set(nums[i], i) and continue.
Flow Diagram

JavaScript Solution

Code

function twoSum(nums, target) {
  const seen = new Map(); // value → index
  for (let i = 0; i < nums.length; i++) {
    const complement = target - nums[i];
    if (seen.has(complement)) {
      return [seen.get(complement), i];
    }
    seen.set(nums[i], i);
  }
  return []; // guaranteed to find a solution
}

// Test
console.log(twoSum([2, 7, 11, 15], 9));  // [0, 1]
console.log(twoSum([3, 2, 4], 6));        // [1, 2]
Time: O(n) Space: O(n)

House Robber

Find the maximum money you can rob from houses without triggering the alarm — you cannot rob two adjacent houses.

DifficultyMedium
Pattern1-D DP
CompaniesAmazon · Microsoft · Google
LanguageJavaScript (ES2020+)
Problem Statement

Description

Given an array nums of non-negative integers representing money in each house, return the maximum amount you can rob without robbing two adjacent houses.

Example: nums = [2,7,9,3,1] → 12 (rob houses 0, 2, 4: 2+9+1=12).

Approach

Core Insight & Strategy

Core insight: At each house you make a binary choice: rob it (gain nums[i], but you must skip house i−1) or skip it (keep the best up to i−1). dp[i] = max(dp[i−1], dp[i−2] + nums[i]). Only two previous values needed.

Step-by-step

  1. Handle edge cases: empty → 0; one house → nums[0].
  2. Init prev2 = nums[0], prev1 = Math.max(nums[0], nums[1]).
  3. For i from 2 to n−1: [prev2, prev1] = [prev1, Math.max(prev1, prev2 + nums[i])].
  4. Return prev1.
Flow Diagram

DP trace — nums=[2,7,9,3,1]

nums = [2, 7, 9, 3, 1] idx: 0 1 2 3 4 dp[0] = 2 dp[1] = max(2, 7) = 7 dp[2] = max(dp[1]=7, dp[0]+9=11) = 11 ← rob house 2 dp[3] = max(dp[2]=11, dp[1]+3=10) = 11 ← skip house 3 dp[4] = max(dp[3]=11, dp[2]+1=12) = 12 ← rob house 4 Answer: 12 (houses 0+2+4: 2+9+1)
JavaScript Solution

Code

function rob(nums) {
  if (!nums.length) return 0;
  if (nums.length === 1) return nums[0];
  let prev2 = nums[0];
  let prev1 = Math.max(nums[0], nums[1]);
  for (let i = 2; i < nums.length; i++) {
    [prev2, prev1] = [prev1, Math.max(prev1, prev2 + nums[i])];
  }
  return prev1;
}

console.log(rob([1, 2, 3, 1]));   // 4
console.log(rob([2, 7, 9, 3, 1])); // 12
Time: O(n) Space: O(1)

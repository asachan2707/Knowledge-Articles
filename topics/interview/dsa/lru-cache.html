LRU Cache

Design an LRU cache with O(1) get and put by combining a doubly linked list (order) with a hash map (lookup).

DifficultyMedium
PatternDoubly Linked List + Hash Map
CompaniesGoogle · Meta · Amazon · Microsoft
LanguageJavaScript (ES2020+)
Problem Statement

Description

Design a data structure that follows the Least Recently Used cache eviction policy. Implement get(key) returning the value (or -1 if absent) and put(key, value). Both must run in O(1). The cache has a fixed capacity; when full, evict the least recently used item.

Example: LRUCache(2) → put(1,1) → put(2,2) → get(1)=1 → put(3,3) [evicts key 2] → get(2)=-1 → get(3)=3.

Approach

Core Insight & Strategy

Core insight: A Map alone gives O(1) lookup but cannot efficiently track recency order. A doubly linked list (DLL) maintains order in O(1) insert/remove. Combine them: the Map stores key→node for O(1) access; the DLL keeps nodes ordered from MRU (tail) to LRU (head). On every access, move the node to the tail. On eviction, remove the node at the head.

Step-by-step

  1. Create dummy head and tail sentinel nodes connected to each other.
  2. _remove(node): unlink node from its neighbours.
  3. _insertTail(node): insert node just before tail sentinel.
  4. get(key): if key in map, remove node and re-insert at tail (mark as recently used), return value. Else return -1.
  5. put(key, value): if key exists, remove old node. Create new node, insert at tail, add to map. If map.size > capacity, remove head.next (LRU) from both DLL and map.
Flow Diagram

Operation trace — capacity=2

Doubly linked list: [head] ↔ ... ↔ [tail] MRU is near tail, LRU is near head. map: key → node put(1,1): [head] ↔ {1:1} ↔ [tail] map={1} put(2,2): [head] ↔ {1:1} ↔ {2:2} ↔ [tail] map={1,2} get(1): move node-1 to tail: [head] ↔ {2:2} ↔ {1:1} ↔ [tail] return 1 put(3,3): capacity=2, must evict LRU LRU = head.next = node-2 → remove it insert node-3 at tail: [head] ↔ {1:1} ↔ {3:3} ↔ [tail] map={1,3} get(2): key 2 not in map → return -1 get(3): move node-3 to tail → return 3
JavaScript Solution

Code

class LRUNode {
  constructor(key, val) {
    this.key = key; this.val = val;
    this.prev = this.next = null;
  }
}

class LRUCache {
  constructor(capacity) {
    this.cap = capacity;
    this.map = new Map();           // key → node
    this.head = new LRUNode(0, 0);  // dummy LRU sentinel
    this.tail = new LRUNode(0, 0);  // dummy MRU sentinel
    this.head.next = this.tail;
    this.tail.prev = this.head;
  }

  _remove(node) {
    node.prev.next = node.next;
    node.next.prev = node.prev;
  }

  _insertTail(node) {
    node.prev = this.tail.prev;
    node.next = this.tail;
    this.tail.prev.next = node;
    this.tail.prev = node;
  }

  get(key) {
    if (!this.map.has(key)) return -1;
    const node = this.map.get(key);
    this._remove(node);
    this._insertTail(node);
    return node.val;
  }

  put(key, val) {
    if (this.map.has(key)) this._remove(this.map.get(key));
    const node = new LRUNode(key, val);
    this._insertTail(node);
    this.map.set(key, node);
    if (this.map.size > this.cap) {
      const lru = this.head.next;
      this._remove(lru);
      this.map.delete(lru.key);
    }
  }
}

const cache = new LRUCache(2);
cache.put(1, 1); cache.put(2, 2);
console.log(cache.get(1));    // 1
cache.put(3, 3);
console.log(cache.get(2));    // -1 (evicted)
console.log(cache.get(3));    // 3
Time: O(1) for get and put Space: O(capacity)

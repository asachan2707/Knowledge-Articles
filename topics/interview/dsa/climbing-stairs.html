Climbing Stairs

Count distinct ways to climb n stairs taking 1 or 2 steps — a Fibonacci recurrence in disguise.

DifficultyEasy
Pattern1-D DP / Fibonacci
CompaniesAmazon · Google · Apple · Microsoft
LanguageJavaScript (ES2020+)
Problem Statement

Description

You are climbing a staircase with n steps. Each time you can climb 1 or 2 steps. How many distinct ways can you reach the top?

Example: n = 5 → 8.

Approach

Core Insight & Strategy

Core insight: To reach step i, you came from either step i−1 (1-step) or step i−2 (2-step). So dp[i] = dp[i−1] + dp[i−2] — the Fibonacci sequence. We only ever need the last two values, so space is O(1).

Step-by-step

  1. Handle base cases: n <= 2 return n.
  2. Init prev2 = 1, prev1 = 2.
  3. For i = 3 to n: [prev2, prev1] = [prev1, prev1 + prev2].
  4. Return prev1.
Flow Diagram

DP table — n=5

dp[i] = ways to reach step i dp[1] = 1 dp[2] = 2 (base cases) dp[3] = dp[2] + dp[1] = 2 + 1 = 3 dp[4] = dp[3] + dp[2] = 3 + 2 = 5 dp[5] = dp[4] + dp[3] = 5 + 3 = 8 Ways to reach step 5: 1-1-1-1-1 2-1-1-1 1-2-1-1 1-1-2-1 1-1-1-2 2-2-1 2-1-2 1-2-2 Total: 8 ✓ Space optimisation: only prev2 and prev1 needed → O(1) space.
JavaScript Solution

Code

function climbStairs(n) {
  if (n <= 2) return n;
  let prev2 = 1, prev1 = 2;
  for (let i = 3; i <= n; i++) {
    [prev2, prev1] = [prev1, prev1 + prev2];
  }
  return prev1;
}

console.log(climbStairs(2)); // 2
console.log(climbStairs(3)); // 3
console.log(climbStairs(5)); // 8
Time: O(n) Space: O(1)

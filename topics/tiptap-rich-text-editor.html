Building a Feature-Rich Text Editor with Tiptap and React

A complete project guide for building a rich text editor with Tiptap v2 and React — covering setup, core extensions, custom extensions, toolbar components, collaboration-ready output formats, and real-world patterns for editor state management, image uploads, and AI integration.

CategoryProject
StackTiptap v2 · React · TypeScript · ProseMirror

Tiptap is a ProseMirror wrapper that makes building rich text editors in React practical. ProseMirror is powerful but demands schema expertise; Tiptap abstracts the schema complexity with a composable extension system. You get the full power of ProseMirror (collaborative editing, custom nodes, marks) with an API that doesn't require a three-day learning curve.

Why Tiptap over alternatives: Quill (old, limited extension model), Draft.js (Facebook's, deprecated in spirit), Slate (flexible but low-level), ProseMirror (full power, steep curve). Tiptap gives you ProseMirror's power with React-native ergonomics and a rich official extension ecosystem.

// Editor.tsx — minimal working editor
import { useEditor, EditorContent } from '@tiptap/react';
import StarterKit from '@tiptap/starter-kit';
import Placeholder from '@tiptap/extension-placeholder';

interface EditorProps {
  content?: string;
  onChange?: (html: string) => void;
  placeholder?: string;
}

export function Editor({ content = '', onChange, placeholder = 'Start writing...' }: EditorProps) {
  const editor = useEditor({
    extensions: [
      StarterKit,  // bold, italic, heading, code, lists, blockquote, horizontal rule
      Placeholder.configure({ placeholder }),
    ],
    content,
    onUpdate: ({ editor }) => onChange?.(editor.getHTML()),
  });

  return ;
}
Core extensions

What StarterKit includes and what to add

StarterKit (included)

Bold, italic, strike, code, headings 1–6, paragraph, bullet list, ordered list, blockquote, code block, horizontal rule, undo/redo

Link

Clickable links with open-on-click, auto-link detection, and href editing

Image

Inline images with optional resize handles (via extension-image-resize)

Table

Full table support with row/column add/remove, cell merging

Mention

@mention with dropdown suggestion list — for users, tags, or any entity

TaskList

Checkable task items — GitHub-flavored checkbox lists

CodeBlock Lowlight

Syntax-highlighted code blocks via lowlight (highlight.js)

Collaboration

// Extended editor with common feature set
import { useEditor, EditorContent } from '@tiptap/react';
import StarterKit       from '@tiptap/starter-kit';
import Link             from '@tiptap/extension-link';
import Image            from '@tiptap/extension-image';
import TaskList         from '@tiptap/extension-task-list';
import TaskItem         from '@tiptap/extension-task-item';
import Placeholder      from '@tiptap/extension-placeholder';
import { common, createLowlight } from 'lowlight';
import CodeBlockLowlight from '@tiptap/extension-code-block-lowlight';

const lowlight = createLowlight(common);

export function RichEditor({ content, onChange }) {
  const editor = useEditor({
    extensions: [
      StarterKit.configure({ codeBlock: false }), // disable to use CodeBlockLowlight instead
      Link.configure({ openOnClick: false, autolink: true }),
      Image.configure({ inline: false, allowBase64: true }),
      TaskList,
      TaskItem.configure({ nested: true }),
      Placeholder.configure({ placeholder: 'Start writing...' }),
      CodeBlockLowlight.configure({ lowlight }),
    ],
    content,
    onUpdate: ({ editor }) => onChange?.(editor.getHTML()),
    editorProps: {
      attributes: { class: 'prose max-w-none focus:outline-none min-h-[200px]' },
    },
  });

  return ;
}

interface ToolbarProps { editor: Editor | null; }

export function Toolbar({ editor }: ToolbarProps) {
  if (!editor) return null;

  return (
    

       editor.chain().focus().toggleBold().run()}
        active={editor.isActive('bold')}
        title="Bold (Cmd+B)"
      >B

       editor.chain().focus().toggleItalic().run()}
        active={editor.isActive('italic')}
        title="Italic (Cmd+I)"
      >I

      {/* Heading selector */}
      {[1, 2, 3].map(level => (
         editor.chain().focus().toggleHeading({ level }).run()}
          active={editor.isActive('heading', { level })}
        >H{level}
      ))}

       editor.chain().focus().toggleBulletList().run()}
        active={editor.isActive('bulletList')}
      >• List

       {
          const url = window.prompt('URL:');
          if (url) editor.chain().focus().setLink({ href: url }).run();
        }}
        active={editor.isActive('link')}
      >Link

       editor.chain().focus().toggleCodeBlock().run()}
        active={editor.isActive('codeBlock')}
      >Code

      {/* Undo / Redo */}
       editor.chain().focus().undo().run()}
        disabled={!editor.can().undo()}
      >↩
       editor.chain().focus().redo().run()}
        disabled={!editor.can().redo()}
      >↪
    

  );
}

Image uploads

Handling image uploads with presigned URLs

// Custom image upload extension
import { Plugin } from '@tiptap/pm/state';
import Image from '@tiptap/extension-image';

const ImageWithUpload = Image.extend({
  addProseMirrorPlugins() {
    return [
      new Plugin({
        props: {
          handleDOMEvents: {
            drop(view, event) {
              const files = event.dataTransfer?.files;
              if (!files?.length) return false;

              Array.from(files)
                .filter(f => f.type.startsWith('image/'))
                .forEach(async (file) => {
                  // 1. Upload to your storage
                  const url = await uploadImage(file);
                  // 2. Insert into editor at drop position
                  const { schema } = view.state;
                  const node = schema.nodes.image.create({ src: url });
                  const pos  = view.posAtCoords({ left: event.clientX, top: event.clientY })?.pos;
                  if (pos != null) {
                    const tr = view.state.tr.insert(pos, node);
                    view.dispatch(tr);
                  }
                });
              return true;
            },
          },
          // Also handle paste
          handlePaste(view, event) {
            const items = event.clipboardData?.items ?? [];
            for (const item of items) {
              if (item.type.startsWith('image/')) {
                item.getAsFile() && uploadImage(item.getAsFile()!).then(url => {
                  const { schema, tr, selection } = view.state;
                  const node = schema.nodes.image.create({ src: url });
                  view.dispatch(tr.replaceSelectionWith(node));
                });
                return true;
              }
            }
            return false;
          },
        },
      }),
    ];
  },
});

async function uploadImage(file: File): Promise {
  const { uploadUrl, publicUrl } = await fetch('/api/upload-url', {
    method: 'POST',
    body: JSON.stringify({ filename: file.name, contentType: file.type }),
    headers: { 'Content-Type': 'application/json' },
  }).then(r => r.json());

  await fetch(uploadUrl, { method: 'PUT', body: file, headers: { 'Content-Type': file.type } });
  return publicUrl;
}

// AI writing assistant integration
async function aiContinue(editor) {
  const currentText = editor.getText(); // plain text version
  const response    = await fetch('/api/ai-continue', {
    method: 'POST',
    body: JSON.stringify({ text: currentText }),
    headers: { 'Content-Type': 'application/json' },
  });
  const reader = response.body!.getReader();
  const decoder = new TextDecoder();

  // Stream AI output directly into editor
  const insertionPos = editor.state.selection.to;
  let buffer = '';

  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    buffer += decoder.decode(value);
    editor.chain().focus().insertContentAt(insertionPos + buffer.length - 1, buffer.slice(-1)).run();
  }
}

// Word count / reading time display
function EditorStats({ editor }) {
  if (!editor) return null;
  const words   = editor.getText().split(/\s+/).filter(Boolean).length;
  const minutes = Math.max(1, Math.round(words / 200));
  return {words} words · {minutes} min read;
}

Key Takeaway

Tiptap's extension model is its biggest strength: you get a fully functional editor from StarterKit in 10 lines, then add only what you need. The editor.chain().focus() API is how you drive formatting from toolbar buttons. For production use: store content as JSON (lossless round-trip), handle image uploads via the drop/paste Plugin hooks, and expose plain text output for any AI integration. The collaboration extension (Y.js) is available when you need real-time multi-user editing, backed by the Hocuspocus server.

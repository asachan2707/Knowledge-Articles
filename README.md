# Learning

A simple static HTML project for documenting concepts and building a personal knowledge base.

## Structure

- `index.html` - homepage for topic navigation
- `styles.css` - shared styling for all pages
- `topics/` - one HTML file per concept or topic

## How to add a topic

1. Create a new HTML file in `topics/`
2. Reuse the structure from `topics/getting-started.html`
3. Add a link to the new topic in `index.html`

## Open in browser

You can open `index.html` directly in a browser to read the content.

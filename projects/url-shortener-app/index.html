URL Shortener Demo Project

This is the runnable companion app for the URL shortener system design topic. It includes a React frontend, a Node backend, seeded in-memory data, and automatic frontend fallback to mock data when the backend is not running.

Use this project to connect architecture thinking with hands-on implementation. The frontend demonstrates the product flow, while the backend shows a simple but real service design with create, list, redirect, and analytics endpoints.

Run the project

Commands

cd Learning/projects/url-shortener-app
npm run setup
npm run dev

Frontend runs on http://localhost:5173 and backend runs on http://localhost:4000.

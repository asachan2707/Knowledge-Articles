# BuildRightLogistics

This project contains a complete architecture package for the BuildRight Logistics Salesforce transformation case study.

## Document Set

- `docs/BuildRight-Logistics-HLD.md`: Primary high-level design document with architecture, security, integration, LDV, migration, sharing, and deployment strategy.
- `docs/LLD-Artifact-Catalog.md`: Low-level artifact inventory with object model details, dedup logic, interface contracts, and deployment runbook.

## Why this package exists

The objective is to provide a presentation-ready design that:

- addresses all mandatory architecture and migration concerns from the case study,
- corrects common anti-patterns in large-scale Salesforce implementations,
- includes concrete low-level implementation guidance so delivery teams can execute without ambiguity.
